| あ (a) | 像「啊」但嘴張小一點 |

## 五句最常用日文旅行用語

### 1. すみません (Sumimasen)

**日文：** すみません  
**羅馬拼音：** su-mi-ma-sen  
**中文：** 不好意思 / 對不起 / 打擾一下  
**用途：** 萬用金句

#### 發音拆解

```
す (su) - 像「司」但不捲舌，嘴巴放鬆
み (mi) - 像「咪」
ま (ma) - 像「嗎」
せ (se) - 像「謝」但不捲舌
ん (n) - 鼻音，像英文的 n
```

**發音技巧：**
- 重量平均分佈在前四個音節
- 語尾「sen」要輕輕收住，不要拖長
- 語調：`su-MI-ma-sen`（第二音節略重）

**使用場景：**
- 引起服務人員注意（點餐、結帳）
- 向路人問路
- 不小心撞到人
- 道歉

#### 例句

```
すみません、トイレはどこですか？
(Sumimasen, toire wa doko desu ka?)
不好意思，請問廁所在哪裡？
```

**例句發音拆解：**
```
すみません (su-mi-ma-sen) — 不好意思
、（停頓）
トイレ (to-i-re) — 廁所（外來語，toe-ray）
は (wa) — 助詞，讀作 wa
どこ (do-ko) — 哪裡
です (de-su) — 是
か (ka) — 疑問助詞
```

**學習要點：**
- 最實用的一句，建議優先記住
- 語氣溫和，適用於各種場合
- 可單獨使用，就能應對 80% 的社交需求

---

### 2. ありがとうございます (Arigatou gozaimasu)

**日文：** ありがとうございます  
**羅馬拼音：** a-ri-ga-tou go-za-i-ma-su  
**中文：** 非常感謝  
**用途：** 表達感謝

#### 發音拆解

```
あ (a) - 啊
り (ri) - 像「李」但舌尖輕點上顎
が (ga) - 像「嘎」
と (to) - 像「偷」但短促
う (ou) - 像長音「喔」

ご (go) - 像「溝」
ざ (za) - 像「雜」
い (i) - 一
ま (ma) - 嗎
す (su) - 司
```

**發音技巧：**
- 「ありがとう」= arigatou（謝謝的非正式說法）
- 「ございます」= gozaimasu（禮貌後綴）
- 整句連起來：`a-ri-ga-TOU go-ZA-i-ma-su`
- 說的時候可以微微鞠躬

**使用場景：**
- 接受服務後（結帳、點餐完成）
- 有人讓座、幫忙
- 離開商店時

#### 例句

```
ありがとうございます！
(Arigatou gozaimasu!)
非常感謝！
```

**學習要點：**
- 「ありがとう」是較隨意的說法
- 「ありがとうございます」是禮貌完整版，旅行建議用這個
- 搭配鞠躬更道地

### 3. これをお願いします (Kore o onegaishimasu)

**日文：** これをお願いします  
**羅馬拼音：** ko-re o o-ne-gai-shi-ma-su  
**中文：** 請給我這個  
**用途：** 點餐、購物

#### 發音拆解

```
こ (ko) - 像「摳」但短促
れ (re) - 像「雷」但舌尖輕彈
を (o) - 助詞，讀作 o（不是 wo）

お (o) - 禮貌前綴
ね (ne) - 像「呢」
が (ga) - 嘎
い (i) - 一
し (shi) - 像「希」
ま (ma) - 嗎
す (su) - 司
```

**發音技巧：**
- 「これ」= ko-re（這個）
- 「お願いします」= o-ne-gai-shi-ma-su（拜託了）
- 重點在「onegai」這個音，要清楚說出

**使用場景：**
- 餐廳點餐（指著菜單）
- 購買商品（指著商品）
- 請求服務

#### 例句

```
これをお願いします。
(Kore o onegaishimasu.)
請給我這個。
```

**延伸用法：**
```
これとこれをお願いします。
(Kore to kore o onegaishimasu.)
請給我這個和這個。

お水をお願いします。
(Omizu o onegaishimasu.)
請給我水。
```

**學習要點：**
- 「これ」= 這個，用手指即可
- 「お願いします」是禮貌的請求用語
- 比亂唸菜名更實用

---

### 4. 〜はどこですか？ (~ wa doko desu ka?)

**日文：** 〜はどこですか？  
**羅馬拼音：** wa do-ko de-su ka  
**中文：** 請問〜在哪裡？  
**用途：** 問路

#### 發音拆解

```
は (wa) - 助詞，讀作 wa（不是 ha）
ど (do) - 像「都」
こ (ko) - 像「摳」
で (de) - 像「爹」
す (su) - 像「司」
か (ka) - 像「卡」
```

**發音技巧：**
- 語調上揚（疑問句）
- 「doko desu ka」要連貫
- 把「〜」替換成你想問的地點

**使用場景：**
- 廁所、車站、景點
- 迷路時

#### 例句

```
駅はどこですか？
(Eki wa doko desu ka?)
請問車站在哪裡？

トイレはどこですか？
(Toire wa doko desu ka?)
請問廁所在哪裡？

コンビニはどこですか？
(Konbini wa doko desu ka?)
請問便利商店在哪裡？
```

**常用詞彙：**

| 場所 | 日文 | 發音 | 發音說明 |
|------|------|------|----------|
| 車站 | 駅 | eki | 像「e-key」連讀 |
| 廁所 | トイレ | toire | toe-ray |
| 便利商店 | コンビニ | konbini | cone-bee-knee |
| 藥妝店 | 薬局 | yakkyoku | yak-kyo-ku |
| 免稅店 | 免税店 | menzeiten | men-zay-ten |

### 5. 英語を話せますか？ (Eigo o hanasemasu ka?)

**日文：** 英語を話せますか？  
**羅馬拼音：** ei-go o ha-na-se-ma-su ka  
**中文：** 請問你會說英文嗎？  
**用途：** 語言求助

#### 發音拆解

```
英 (ei) - 像「A」的長音
語 (go) - 像「溝」

を (o) - 助詞

話 (ha) - 像「哈」
せ (se) - 像「謝」
ま (ma) - 像「嗎」
す (su) - 像「司」
か (ka) - 疑問助詞
```

**發音技巧：**
- 「英語」= ei-go（英文）
- 「話せます」= ha-na-se-ma-su（能說）
- 「か」表示疑問，語調上揚

**使用場景：**
- 溝通困難時
- 需要詳細說明時
- 緊急情況

#### 例句

```
すみません、英語を話せますか？
(Sumimasen, Eigo o hanasemasu ka?)
不好意思，請問你會說英文嗎？
```

**相關用語：**
```
中国語を話せますか？
(Chuugokugo o hanasemasu ka?)
請問你會說中文嗎？

メニューは英語ありますか？
(Menyuu wa eigo arimasu ka?)
請問有英文菜單嗎？
```

**學習要點：**
- 日本年輕人較敢講英文
- 觀光區服務人員多具備基本英語能力
- 搭配翻譯 App 更順暢

## 學習建議

### 記憶順序
1. **第一優先：** すみません — 萬用開場
2. **第二優先：** ありがとうございます — 必備禮貌
3. **第三優先：** これをお願いします — 吃飯購物必備
4. **第四優先：** 〜はどこですか？ — 迷路救星
5. **第五優先：** 英語を話せますか？ — 備用方案

### 練習方法
1. 每天唸 3 次，連續一週
2. 用手機錄音，對照線上 TTS 修正發音
3. 想像場景，實際說出來（搭配手勢）
4. 到日本前兩天，早起複習一次

### 實戰技巧
- 說完日文後，對方可能回覆日文，這時候微笑點頭即可
- 手勢 + 單字比完整句子更有效
- 善用 Google Translate 即時翻譯功能
- 日本人很有耐心，不用害怕
