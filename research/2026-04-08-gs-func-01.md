# GOOGLEFINANCE 函數深入研究報告

**日期：** 2026-04-08  
**作者：** Tina  
**目標：** 研究 Google Sheets `GOOGLEFINANCE` 函數的能力，用於收集 ETF 歷史資料

---

## 一、函數語法

```google_sheets
=GOOGLEFINANCE(ticker, [attribute], [start_date], [end_date|num_days], [interval])
```

### 參數說明

| 參數 | 必填/選填 | 說明 |
|------|----------|------|
| `ticker` | **必填** | 證券代碼，建議使用「交易所:代碼」格式（如 `NYSEARCA:VOO`） |
| `attribute` | 選填 | 要獲取的資料屬性，預設為 `price` |
| `start_date` | 選填 | 歷史資料起始日期 |
| `end_date` | 選填 | 歷史資料結束日期（可與 `num_days` 二選一） |
| `num_days` | 選填 | 從起始日期開始的天數 |
| `interval` | 選填 | 資料頻率：`DAILY` 或 `1`（每日）、`WEEKLY` 或 `7`（每週） |

---

## 二、可獲取的資料屬性

### 2.1 即時資料屬性（Real-time Data）

當**不指定日期參數**時，可以獲取以下即時資料：

| 屬性 | 說明 |
|------|------|
| `price` | 即時價格（延遲最多 20 分鐘） |
| `priceopen` | 開盤價 |
| `high` | 當日最高價 |
| `low` | 當日最低價 |
| `volume` | 當日交易量 |
| `marketcap` | 市值 |
| `tradetime` | 最後交易時間 |
| `datadelay` | 資料延遲時間 |
| `volumeavg` | 平均日交易量 |
| `pe` | 本益比（P/E ratio） |
| `eps` | 每股盈餘 |
| `high52` | 52 週最高價 |
| `low52` | 52 週最低價 |
| `change` | 較前日收盤價的價格變動 |
| `changepct` | 較前日收盤價的變動百分比 |
| `beta` | Beta 值 |
| `closeyest` | 前一日收盤價 |
| `shares` | 流通股數 |
| `currency` | 計價貨幣 |

### 2.2 歷史資料屬性（Historical Data）

當**指定日期參數**後，只能獲取以下歷史資料：

| 屬性 | 說明 |
|------|------|
| `open` | 開盤價 |
| `close` | 收盤價 |
| `high` | 最高價 |
| `low` | 最低價 |
| `volume` | 交易量 |
| `all` | 以上所有屬性（一次獲取全部） |

> ⚠️ **重要限制：** 一旦指定任何日期參數，請求即被視為「歷史資料」，只能使用歷史屬性。無法同時獲取即時與歷史資料。

## 四、日期格式說明

### 4.1 日期格式選項

1. **字串格式：** `"MM/DD/YYYY"` 或 `"DD/MM/YYYY"`（視試算表設定）
2. **DATE 函數：** `DATE(年, 月, 日)`
3. **相對日期：** `TODAY()-N`（從今天往前 N 天）

### 4.2 日期處理注意事項

- Google 會將傳入的日期視為 **UTC 中午 12:00**
- 在此時間之前收盤的交易所，日期可能會被移動一天
- 如果只提供 `start_date` 而未提供 `end_date`，只會返回該單日資料

## 五、以 VOO 為例的實際應用

### 5.1 VOO ETF 基本資訊

- **名稱：** Vanguard S&P 500 ETF
- **交易所：** NYSE Arca
- **代碼：** VOO
- **發行日期：** 2010-09-07
- **計價貨幣：** USD

### 5.2 獲取即時價格

```google_sheets
=GOOGLEFINANCE("NYSEARCA:VOO", "price")
```

### 5.3 獲取完整即時資訊（多屬性分別查詢）

```google_sheets
=GOOGLEFINANCE("NYSEARCA:VOO", "high52")    // 52週最高價
=GOOGLEFINANCE("NYSEARCA:VOO", "low52")     // 52週最低價
=GOOGLEFINANCE("NYSEARCA:VOO", "pe")        // 本益比
=GOOGLEFINANCE("NYSEARCA:VOO", "currency")  // 計價貨幣
```

### 5.4 獲取最近 30 天歷史資料（收盤價）

```google_sheets
=GOOGLEFINANCE("NYSEARCA:VOO", "close", TODAY()-30, TODAY(), "DAILY")
```

### 5.5 獲取從發行日至今的所有歷史資料

```google_sheets
=GOOGLEFINANCE("NYSEARCA:VOO", "all", DATE(2010,9,7), TODAY(), "DAILY")
```

> 📌 **公式說明：**
> - `all` 屬性會返回 Date、Open、High、Low、Close、Volume 六欄資料
> - 從 2010-09-07 發行日開始至今約 15 年，每日資料約 **3,900 筆**（不含週末假日）
> - 資料量較大，可能影響試算表效能

### 5.6 獲取特定年份歷史資料

```google_sheets
// 2024 年全年資料
=GOOGLEFINANCE("NYSEARCA:VOO", "close", DATE(2024,1,1), DATE(2024,12,31), "DAILY")

// 2023 年全年資料（每週）
=GOOGLEFINANCE("NYSEARCA:VOO", "all", DATE(2023,1,1), DATE(2023,12,31), "WEEKLY")
```

### 5.7 獲取單日歷史價格

```google_sheets
// 獲取 2020-03-23（疫情低點）的收盤價
=INDEX(GOOGLEFINANCE("NYSEARCA:VOO", "close", DATE(2020,3,23)), 2, 2)
```

> 💡 **技巧：** 使用 `INDEX(..., 2, 2)` 只返回價格值（不包含標題列）

## 六、重要限制與注意事項

### 6.1 資料來源限制

| 限制 | 說明 |
|------|------|
| **不支援股利資料** | 無法獲取股利發放歷史、股息收益率 |
| **不支援國際交易所** | 僅支援部分交易所（主要為美國市場） |
| **僅英文版本** | 函數僅在英文版 Google Sheets 中可用 |
| **資料延遲** | 即時價格最多延遲 20 分鐘 |

### 6.2 API 存取限制

```
⚠️ 無法透過 Google Sheets API 或 Apps Script 獲取歷史資料
```

如果嘗試透過 API 存取歷史資料，會返回 `#N/A` 錯誤。這意味著：
- 必須在 Google Sheets 介面中直接使用函數
- 無法使用 Apps Script 自動化提取歷史資料

### 6.3 效能考量

- 大量 `GOOGLEFINANCE` 公式可能導致 `#N/A` 錯誤
- 長時間跨度（如 15 年每日資料）會產生大量儲存格
- 建議分批獲取或使用每週資料減少資料量

### 6.4 資料可靠性

```
This data is not for financial industry professional use.
```

- 資料僅供參考，不適合作為專業金融決策依據
- 資料可能不完整或有錯誤
- Google 不保證資料的準確性和即時性

## 七、進階應用範例

### 7.1 批次獲取多個 ETF 資料

假設 A 欄列有 ETF 代碼清單：

```
A2: NYSEARCA:VOO
A3: IVV
A4: SPY
```

B2 公式（下拉套用）：
```google_sheets
=GOOGLEFINANCE(A2, "close", TODAY()-30, TODAY())
```

### 7.2 使用 ARRAYFORMULA 批次查詢即時價格

```google_sheets
=ARRAYFORMULA(GOOGLEFINANCE(A2:A10, "price"))
```

> ⚠️ `ARRAYFORMULA` 與歷史資料查詢不相容，僅適用於即時屬性

### 7.3 結合 SPARKLINE 繪製走勢圖

```google_sheets
=SPARKLINE(GOOGLEFINANCE("NYSEARCA:VOO", "close", TODAY()-30, TODAY()), 
            {"charttype", "line"; "color", "blue"})
```

### 7.4 計算投資報酬率

```google_sheets
// 計算 30 天報酬率
=(GOOGLEFINANCE("NYSEARCA:VOO", "price") - 
  INDEX(GOOGLEFINANCE("NYSEARCA:VOO", "close", TODAY()-30), 2, 2)) / 
  INDEX(GOOGLEFINANCE("NYSEARCA:VOO", "close", TODAY()-30), 2, 2)
```

## 八、替代方案

如果 `GOOGLEFINANCE` 無法滿足需求，可考慮：

- 非官方 API，可獲取更豐富的歷史資料
- 支援股利、分割等資訊
- 可透過 Co:json 或其他 ETL 工具匯入

### 8.2 其他金融資料來源

## 九、總結

### 適合使用 GOOGLEFINANCE 的場景

✅ 即時價格監控  
✅ 52 週高低價查詢  
✅ 近期歷史價格（數月至一年內）  
✅ 簡單的投資組合追蹤  
✅ 每日/每週價格趨勢分析  

### 不適合使用 GOOGLEFINANCE 的場段

❌ 需要股利歷史資料  
❌ 需要 API 自動化存取  
❌ 長期歷史資料（超過 10 年）  
❌ 專業金融決策使用  
❌ 高頻率即時資料
